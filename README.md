# Drug Discovery AI

A comprehensive drug discovery platform with AI-powered predictions, disease analysis, and toxicity assessment.

## Features

- AI Drug Prediction
- Disease Analyzer
- Drug Search
- Toxicity Predictor
- User Authentication
- History Tracking

## Tech Stack

- Backend: Python, FastAPI
- Frontend: React, Tailwind CSS
- Database: SQLite
- ML Models: Custom predictors

## Getting Started

1. Clone the repository
2. Install backend dependencies: `pip install -r backend/requirements.txt`
3. Install frontend dependencies: `npm install` in frontend/
4. Run the backend: `uvicorn backend.main:app --reload`
5. Run the frontend: `npm start` in frontend/
